# Tableau – Overview

---

## What is Tableau?
**Tableau** is a **Business Intelligence (BI) and Data Visualization tool** used to analyze data and create **interactive dashboards, reports, and charts**.

Tableau uses a **drag-and-drop interface** to convert raw data into meaningful visual insights.

---

## Tableau Services / Products

### 1. Tableau Desktop
- Used to **create dashboards and reports**
- Connects to live or extracted data
- Installed on a local system
- Mainly used by **data analysts and developers**

### 2. Tableau Server
- Used to **publish, share, and manage dashboards**
- Installed on an organization’s server
- Supports **user access control and security**

### 3. Tableau Online (Tableau Cloud)
- Cloud-based version of Tableau Server
- No infrastructure or installation required
- Managed entirely by Tableau

### 4. Tableau Public
- **Free version** of Tableau
- All dashboards are **publicly accessible**
- Used for learning, portfolios, and storytelling

### 5. Tableau Prep
- Used for **data cleaning and preparation**
- Combine, clean, filter, and transform data
- Similar to **Power Query in Power BI**

### 6. Tableau Mobile
- View dashboards on mobile devices
- Real-time access to reports anywhere

## Tableau vs Excel

| Feature | Tableau | Excel |
|------|--------|-------|
| Purpose | BI & Visualization | Spreadsheet calculations |
| Data Handling | Large datasets | Small to medium datasets |
| Dashboards | Interactive & dynamic | Mostly static |
| Automation | High | Limited |
| Real-time Data | Supported | Limited |
| Coding Required | No | No |

**Summary:**
- **Excel** → Best for calculations and small analysis
- **Tableau** → Best for visual analytics and dashboards

---

## Tableau vs Power BI

| Feature | Tableau | Power BI |
|------|--------|---------|
| Company | Salesforce | Microsoft |
| Visualization | Excellent (Best-in-class) | Good |
| Ease of Use | Very easy for visuals | Easy for Excel users |
| Data Modeling | Limited | Very strong (DAX) |
| Performance | Excellent for large data | Very good |
| Cost | Expensive | Affordable |
| Integration | Multiple data sources | Best with Microsoft tools |
| Learning Curve | Easy | Medium |

**Summary:**
- **Tableau** → Best for **advanced visualization & storytelling**
- **Power BI** → Best for **data modeling & Microsoft ecosystem**

---

## When to Use What?
- **Excel** → Data entry, formulas, quick analysis
- **Power BI** → Microsoft tools, strong data modeling
- **Tableau** → Enterprise dashboards, visual storytelling
